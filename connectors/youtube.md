# <img src="../.gitbook/assets/youtube.svg" alt="" data-size="line"> YouTube

Connect your YouTube channel to Scratch to manage video titles, descriptions, and metadata with AI.

## Authentication

YouTube requires OAuth authentication:

- Click "Connect with YouTube" and sign in with your Google account
- Authorize Scratch to access your YouTube channel

## What you can sync

- Video titles and descriptions
- Video metadata and tags
- Channel information

## Getting started

1. Go to **Data Sources** in Scratch
2. Click **Add Connection** and select YouTube
3. Sign in with Google and authorize access
4. Select the channel you want to work with
5. Create a Workbook from your connection

## Supported field types

| Field | Support |
|-------|---------|
| Title | ✅ Read/Write |
| Description | ✅ Read/Write |
| Category | ✅ Read/Write |
| Default language | ✅ Read/Write |
| Transcript | ✅ Read/Write* |
| Tags | ➡️ Read Only |
| Privacy status | ➡️ Read Only |
| Published date | ➡️ Read Only |
| Video URL | ➡️ Read Only |
| Studio URL | ➡️ Read Only |

*Transcript editing works with draft captions, not auto-generated captions.

## Use cases

- Standardize video titles across your channel
- Improve video descriptions for SEO
- Add consistent calls-to-action to all videos
- Bulk update outdated information
